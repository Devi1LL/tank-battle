# 🎮 坦克大战 Tank Battle

经典坦克大战游戏，Browser-based，纯 HTML / CSS / JavaScript Canvas 实现，灵感来自 **Battle City (NES)**。

## 玩法

| 操作 | 按键 |
|------|------|
| 移动 | `W` `A` `S` `D` |
| 射击 | `Space` / `J` |
| 暂停 | `P` |
| 重新开始 | 点击按钮或 `Enter`（游戏结束后） |

## 特性

- 多层 AI 敌人，难度逐关递增（速度、数量、钢墙）
- 砖墙可破坏，钢墙不可破坏
- 基地（老鹰）保护机制
- 粒子爆炸特效
- 5 个关卡

## 快速开始

直接用浏览器打开 `tank-battle.html` 即可游玩，无需任何依赖。
